This site is built with **Barka** — the progressive Content-as-Code CMS.

## What is Barka?

Barka is a files-first CMS where content lives in Markdown and YAML. You can edit content with any text editor, with AI agents, or through the optional admin UI.

## Getting Started

1. Edit files in `content/` to add pages, articles, and more
2. Configure your site in `config/settings.yaml`
3. Customize the theme in `themes/`
4. Run `barka dev` to preview your changes
5. Run `barka build` to generate a static site
